HOW TO USE
================================================================================

Option 1: Direct Open (Fastest)
  1. Open index.html in any browser
  2. Demo mode enabled by default
  3. Click "Westlake Commerce Center" to see dashboard
  4. All features work without backend

Option 2: Local HTTP Server
  1. cd /sessions/optimistic-nifty-turing/mnt/outputs/cre-deal-underwriting/frontend
  2. python -m http.server 8080
  3. Navigate to http://localhost:8080

Option 3: Connect to Backend
  1. Edit index.html line 47: const DEMO_MODE = false;
  2. Ensure FastAPI backend running at http://localhost:8000
  3. Refresh browser
  4. App will fetch real deals from backend

DEPLOYMENT INSTRUCTIONS
================================================================================

Development:
  1. Open index.html in browser (no server needed for demo)
  2. Or: python -m http.server 8080
  3. Navigate to http://localhost:8080

Staging:
  1. Host on staging server at static URL
  2. Configure API_URL to staging backend
  3. Set DEMO_MODE = false
  4. Test all API integration
  5. Verify CORS headers on backend

Production:
  1. Host on CDN or static hosting
  2. Configure API_URL to production backend
  3. Set DEMO_MODE = false
  4. Enable HTTPS (SSL/TLS)
  5. Verify CORS on backend
  6. Configure rate limiting on backend
  7. Monitor error logs and performance

Optional: Minify for 30% size reduction
  npx html-minifier --minify-js --minify-css index.html > index.min.html
